# Art History Field Guide

A polished, beginner-friendly static website that turns a substantial art-history quick guide into an interactive learning companion.

## What it includes

- Big-picture overview of Western art history
- Clickable timeline explorer from Ancient to Contemporary
- Personalized starting paths for Caravaggio, Rembrandt, and Picasso
- Art appreciation framework for looking at paintings
- Beginner glossary with live search
- Quick self-test quiz
- Curated external learning links

## Local preview

Open `index.html` directly in a browser, or run a tiny local server:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000/
```

## Publishing

This repo is designed to work well on GitHub Pages from the repository root.

## Notes

The site is dependency-light: plain HTML, CSS, and JavaScript, plus Google Fonts.
